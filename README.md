# ReachLog

Smart job outreach tracker. Paste a cold email or LinkedIn message — AI extracts the details, creates a record, and tracks recruiter engagement.

## Tech Stack

- **Backend:** ASP.NET Core 9, EF Core, PostgreSQL
- **Frontend:** Angular 21
- **Infrastructure:** Docker, GitHub Actions CI/CD
